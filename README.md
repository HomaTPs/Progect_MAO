## ✨ Возможности

### Основной функционал
- ✅ **Добавление тренировок** - быстрая запись новых тренировок с указанием даты, типа и длительности
- ✅ **Валидация данных** - автоматическая проверка корректности вводимых данных (формат даты: ГГГГ-ММ-ДД, длительность: положительное число)
- ✅ **Фильтрация записей** - удобный поиск тренировок по типу и дате
- ✅ **Табличное отображение** - все тренировки представлены в удобной таблице с возможностью прокрутки
- ✅ **Управление записями** - добавление, удаление выбранных записей и полная очистка всех данных

### Работа с данными
- 💾 **Автоматическое сохранение** - все изменения мгновенно сохраняются в JSON файл (trainings.json)
- 📂 **Ручное сохранение** - возможность сохранить данные в выбранный пользователем файл
- 📖 **Загрузка из JSON** - импорт тренировок из любого JSON файла через диалоговое окно
- 🎯 **Демо-данные** - при первом запуске автоматически создаются примеры тренировок для демонстрации

### Интерфейс
- 🎨 **Интуитивно понятный GUI** - простой и удобный интерфейс на русском языке
- 🔄 **Автообновление** - таблица автоматически обновляется при любых изменениях (добавление, удаление, фильтрация)
- 🖱️ **Поддержка удаления** - удаление выбранных записей одним кликом с подтверждением
- 📊 **Полосы прокрутки** - удобная навигация по большому количеству записей
- 🎨 **Цветовое выделение кнопок** - разные цвета для различных действий (зеленый - добавить, красный - удалить, синий - фильтр)

## 🛠️ Технологии

- **Python 3.x** - язык программирования
- **Tkinter** - стандартная библиотека для создания графического интерфейса

## 🎮 Использование

### Добавление тренировки
# Пошаговая инструкция по добавлению тренировки:
1. Заполните поля в верхней части окна:
   - Поле "Дата": введите дату в формате ГГГГ-ММ-ДД (например, 2024-12-25)
   - Поле "Тип тренировки": выберите значение из выпадающего списка
   - Поле "Длительность": введите положительное число (целое или дробное)

2. Нажмите кнопку "Добавить тренировку" (зеленая кнопка)

3. Проверьте результат:
   - Тренировка появится в основной таблице
   - Данные автоматически сохранятся в файл trainings.json
   - Появится всплывающее сообщение об успешном добавлении

# Пример правильного заполнения:
Дата: 2024-12-25
Тип тренировки: Бег
Длительность: 30

### Фильтрация данных
# Фильтрация по типу тренировки:
1. В блоке "Фильтрация" найдите выпадающий список "Тип тренировки"
2. Выберите нужный тип из списка (Бег, Плавание, Велосипед, Силовая, Йога, Растяжка)
3. Нажмите кнопку "Применить фильтр" (синяя кнопка)
4. Таблица автоматически обновится, показывая только выбранный тип тренировок

# Фильтрация по дате:
1. В поле "Дата (ГГГГ-ММ-ДД)" введите нужную дату
   Пример: 2024-12-25
2. Нажмите кнопку "Применить фильтр"
3. В таблице останутся только тренировки за указанную дату

# Комбинированная фильтрация:
1. Выберите тип тренировки из списка
2. Введите дату в соответствующее поле
3. Нажмите "Применить фильтр"
4. Будут показаны тренировки, соответствующие обоим условиям

# Сброс фильтров:
- Нажмите кнопку "Сбросить фильтры" (оранжевая кнопка)
- Все фильтры будут очищены
- Таблица покажет все тренировки

### Управление тренировками
# Удаление выбранной тренировки:
1. В таблице кликните мышью на запись, которую хотите удалить
   (Запись выделится синим цветом)
2. Нажмите кнопку "Удалить выбранную" (красная кнопка)
3. Подтвердите удаление в появившемся диалоговом окне
4. Тренировка будет удалена из таблицы и из файла

# Полная очистка всех данных:
1. Нажмите кнопку "Очистить всё" (оранжевая кнопка)
2. Подтвердите действие в диалоговом окне
3. Все тренировки будут удалены без возможности восстановления

# Сохранение данных в JSON файл:
1. Нажмите кнопку "Сохранить в JSON" (зеленая кнопка)
2. Данные автоматически сохранятся в файл trainings.json
3. Появится сообщение об успешном сохранении

# Загрузка данных из JSON файла:
1. Нажмите кнопку "Загрузить из JSON" (синяя кнопка)
2. В открывшемся диалоговом окне выберите JSON файл
3. Нажмите "Открыть"
4. Данные из файла загрузятся в программу
5. Таблица автоматически обновится

### Валидация данных
# Программа автоматически проверяет следующие условия:

# Проверка даты:
- Формат должен быть строго ГГГГ-ММ-ДД
- Примеры правильных дат: 2024-12-25, 2025-01-01, 2024-02-28
- Примеры неправильных дат: 25.12.2024, 12/25/2024, 2024-13-45
- При ошибке появится сообщение: "Неверный формат даты! Используйте ГГГГ-ММ-ДД"

# Проверка длительности:
- Длительность должна быть положительным числом
- Можно использовать целые числа: 30, 45, 60
- Можно использовать дробные числа: 30.5, 45.75
- Примеры неправильных значений: -30, 0, abc, пустое поле
- При ошибке появится сообщение: "Длительность должна быть положительным числом!"

# Проверка обязательных полей:
- Все поля должны быть заполнены
- При пустых полях появится соответствующее сообщение об ошибке
