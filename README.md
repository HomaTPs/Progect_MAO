## ✨ Возможности

### Основной функционал
- ✅ **Добавление тренировок** - быстрая запись новых тренировок с указанием даты, типа и длительности
- ✅ **Валидация данных** - автоматическая проверка корректности вводимых данных (формат даты: ГГГГ-ММ-ДД, длительность: положительное число)
- ✅ **Фильтрация записей** - удобный поиск тренировок по типу и дате
- ✅ **Табличное отображение** - все тренировки представлены в удобной таблице с возможностью прокрутки
- ✅ **Управление записями** - добавление, удаление выбранных записей и полная очистка всех данных

### Работа с данными
- 💾 **Автоматическое сохранение** - все изменения мгновенно сохраняются в JSON файл (trainings.json)
- 📂 **Ручное сохранение** - возможность сохранить данные в выбранный пользователем файл
- 📖 **Загрузка из JSON** - импорт тренировок из любого JSON файла через диалоговое окно
- 🎯 **Демо-данные** - при первом запуске автоматически создаются примеры тренировок для демонстрации

### Интерфейс
- 🎨 **Интуитивно понятный GUI** - простой и удобный интерфейс на русском языке
- 🔄 **Автообновление** - таблица автоматически обновляется при любых изменениях (добавление, удаление, фильтрация)
- 🖱️ **Поддержка удаления** - удаление выбранных записей одним кликом с подтверждением
- 📊 **Полосы прокрутки** - удобная навигация по большому количеству записей
- 🎨 **Цветовое выделение кнопок** - разные цвета для различных действий (зеленый - добавить, красный - удалить, синий - фильтр)

## 🛠️ Технологии

- **Python 3.x** - язык программирования
- **Tkinter** - стандартная библиотека для создания графического интерфейса
- **JSON** - формат хранения данных
- **Datetime** - библиотека для работы с датами и их валидации
- **OS** - работа с файловой системой

## 📥 Установка

- Установленный Python 3.6 или выше

1. **Скачайте файл программы** `main.py`

2. **Запустите приложение** через командную строку или двойным кликом:
```bash
python main.py

- Примечание: Все необходимые библиотеки входят в стандартную поставку Python, дополнительная установка не требуется.
